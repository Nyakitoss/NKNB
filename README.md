# NKNB - Telegram News Bot

Автоматический Telegram-бот для публикации новостных дайджестов с использованием AI Gemini.

## Особенности

- 🤖 Автоматическая генерация новостей через Google Gemini
- 📝 Публикация в заданное время (по умолчанию 09:00)
- 🎯 Настраиваемые темы новостей
- 💾 Сохранение конфигурации в Redis (для Railway)
- 🔒 Проверка прав администратора канала
- 📱 Удобный интерфейс с inline-кнопками

## Установка и запуск

### Локальная разработка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Nyakitoss/NKNB.git
cd NKNB
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните переменные окружения в `.env`:
```
API_ID=your_api_id
API_HASH=your_api_hash
NEWS_BOT_TOKEN=your_bot_token
GEMINI_API_KEY=your_gemini_api_key
USE_REDIS=false
```

5. Запустите бота:
```bash
python news_bot.py
```

### Деплой на Railway

1. Подключите репозиторий GitHub к Railway
2. Добавьте Redis Plugin в Railway проект
3. Установите переменные окружения в Railway:
   - `API_ID`
   - `API_HASH` 
   - `NEWS_BOT_TOKEN`
   - `GEMINI_API_KEY`
   - `USE_REDIS=true`
   - `REDIS_HOST` (автоматически от Railway)
   - `REDIS_PORT` (автоматически от Railway)
   - `REDIS_PASSWORD` (автоматически от Railway)

4. Деплой произойдет автоматически

## Использование

### Команды бота

- `/start` - Начать настройку канала
- `/post_now` - Опубликовать новости немедленно

### Настройка канала

1. Отправьте username канала (например `@my_channel`)
2. Выберите темы новостей через inline-кнопки
3. Нажмите "💾 Сохранить"

### Требования к каналу

- Канал должен быть публичным
- Бот должен быть администратором канала
- У бота должны быть права на отправку сообщений

## Решение проблемы с сохранением данных

На Railway файловая система сбрасывается при каждом деплое. Для решения этой проблемы:

1. **Redis**: Используется как персистентное хранилище
2. **Fallback**: При недоступности Redis используется локальное хранилище
3. **Автоматическое переключение**: Бот автоматически определяет доступность Redis
